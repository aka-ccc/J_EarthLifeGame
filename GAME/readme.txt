1.J地球生中所用到的8個.java檔各包含一個public class
2.欲執行遊戲僅需對Game.java檔案進行編譯並執行
	指令如下：
		javac Game.java
		java Game
3.欲執行J地球生需安裝好java執行環境，包含JDK和JRE及環境變數(在OOPII課程提供的主機(140.127.220.235)上可直接對遊戲進行編譯、執行
4.只要能夠執行，就代表安裝成功了
5.附上各檔案大致內容(詳細內容請見文件)

	Game.java：為《J 地球生》的遊戲主程式檔，遊玩此遊戲時需編譯此檔案並執行。
	GameFrame.java：為《J 地球生》遊戲主視窗，其中會放置遊戲主畫面並針對視窗關閉動作加以設定。
	CheckExit.java：為使用者關閉遊戲時所跳出的確認視窗，目的為確認使用者是否要離開遊戲。
	ScencePanel.java：為《J 地球生》遊戲主畫面，負責顯示遊戲畫面 (含遊戲主畫面、規則說明、遊戲畫面等)。
	Player.java：用以生成玩家角色的初始狀態，並在其內撰寫各式函數以更改、取得角色的狀態。
	PlayAction.java：為卡牌對應動作的運算。
	ComputerPlay.java：為電腦玩家抽牌、出牌的執行，並在其中會呼叫 PlayAction 以完成卡牌對應動作的運算。
	Deck.java：用以載入卡牌的圖片。
